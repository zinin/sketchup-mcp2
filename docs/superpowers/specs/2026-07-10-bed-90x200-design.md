# Кровать 90×200 с ящиками — дизайн модели для SketchUp MCP

Дата: 2026-07-10. Статус: согласовано в brainstorming-сессии.

## 1. Задача

Смоделировать в SketchUp (через MCP) кровать под матрас **2000 × 900 × 200 мм**
с выдвижными ящиками — точную деталировочную модель для последующей постройки:
каждая деталь — отдельная именованная группа с размерами под имеющийся материал,
плюс упрощённая фурнитура (направляющие) для проверки зазоров.

В сцене — **две кровати рядом** (варианты для выбора, каркас идентичный):

| | Кровать A | Кровать B |
|---|---|---|
| Ящики | 2 больших (~150 л каждый) | 3 средних (~100 л каждый) |
| Направляющие | левый ящик — боковые шариковые 750 мм; правый — скрытого монтажа 750 мм | все три — боковые шариковые 750 мм |
| Изголовье | есть, верх на 900 мм от пола | нет |
| Выдвинут для наглядности | левый ящик, на 350 мм | средний ящик, на 350 мм |

## 2. Принятые решения (из сессии вопросов)

- Пользователь: взрослый / гостевая — полный запас прочности, посадка ~58 см.
- Кровать стоит длинной стороной к стене (или в углу) → ящики только с одной
  длинной стороны.
- Конструкция: **панельный короб** (стенки из щита до пола, стиль IKEA Brimnes) —
  выбрана из трёх вариантов (короб / рама на ножках / короб на цоколе).
- Матрас утоплен в короб на 40 мм (бортик удерживает матрас).
- Изножья нет ни в одном варианте.
- Крепёж, ручки и отделку не моделируем (см. §10).

## 3. Конструкция и высотная схема

Передней стенки до пола нет: весь фронт ниже фасадной царги закрывают накладные
фасады ящиков. Направляющие крайних ящиков крепятся к торцевым стенкам короба,
средних — к перегородкам из фанеры 18, которые заодно подпирают настил посередине.

Высоты от пола (одинаковы для A и B):

```
580  верх матраса (матрас 200)
420  верх борта короба (матрас утоплен на 40)
380  верх настила (ламели 20)
360  верх опорных брусков 40×40 и перегородок
290  низ фасадной царги (царга 130)
286  верх фасада ящика (фасад 270)
280  верх коробки ящика (боковина 240)
 40  низ коробки ящика
 16  низ фасада (зазор до пола)
  0  пол — стенки короба стоят на полу
```

Зазоры: фасад ↔ царга и фасад ↔ фасад — 4 мм; верх коробки ящика ↔ низ царги —
10 мм; верх коробки ↔ низ ламелей — 80 мм (запас на прогиб настила).

## 4. Координатная система и компоновка сцены

- **X** — вдоль кровати (0 → 2056), **Y** — поперёк (0 — задняя, «стеночная»
  плоскость; фасады смотрят в +Y), **Z** — вверх от пола (0).
- Origin кровати A — задний левый нижний угол короба в мировом (0, 0, 0).
- Кровать B — та же локальная геометрия со сдвигом **X +2656** (2056 + зазор 600).
- Все размеры в модели — миллиметры (MCP-инструменты принимают мм).

Ключевые плоскости по Y: задняя стенка 0–18; просвет под матрас 18–938 (920);
фасадная плоскость короба 938; накладные фасады 938–956. Полный габарит короба
с фасадами **2056 × 956 × 420**; с изголовьем длина 2074.

## 6. Спецификация — кровать B (3 ящика), дельта от A

- Перегородки две: `B.box.partition_1` X 679–697, `B.box.partition_2`
  X 1358–1376 (размер как у A). Камеры: 18–679, 697–1358, 1376–2038
  (661 / 661 / 662).
- Ящики три, все на боковых направляющих: коробка снаружи **636 × 750 × 240**
  (перед/зад 612 × 240 × 12; дно 622 × 736 × 6), в камерах по центру
  (зазор 12.5 на сторону).
- Фасады три: **680 × 270 × 18**, X 4–684, 688–1368, 1372–2052; Y и Z как у A.
- Изголовья нет. Средний ящик выдвинут на Y +350.
- Остальное (короб, настил, матрас) — идентично A.

## 7. Структура групп в SketchUp

```
Bed_A                          Bed_B
├─ A.box.*   (5 деталей)       ├─ B.box.*   (6 деталей)
├─ A.slats.* (2 бруска + 13)   ├─ B.slats.* (2 бруска + 13)
├─ A.drawer1.* (+slides)       ├─ B.drawer1..3.* (+slides)
├─ A.drawer2.* (+slides)       └─ B.mattress
├─ A.headboard
└─ A.mattress
```

Каждая деталь — группа с именем из таблиц выше; узлы (`Box`, `Slats`,
`Drawer_N`, …) — вложенные группы внутри `Bed_A` / `Bed_B`, чтобы любой узел
можно было скрыть и снять спецификацию через `list_components`.

Материалы (hex): щит/фасады/ламели — сосна `#E6C79A`; фанера — `#F0DDB0`;
направляющие — металлик `#9EA4AB`; матрас — `#D9D9DE`. Обе кровати в одинаковых
материалах.

## 8. Порядок моделирования и верификация

1. Строим через `eval_ruby` параметрическим скриптом (хелпер `make_box` из
   `docs/sketchup-ruby-cookbook.md`; блок констант с размерами в мм) — обе
   кровати создаются одним кодом с параметрами: число ящиков, изголовье,
   тип направляющих. Скрипт подаётся порциями по узлам (короб → настил →
   ящики → изголовье → матрас), каждая порция — одна операция undo.
2. После каждого узла: сверка `bbox_mm` созданных групп с координатами из
   таблиц §5–6.
3. В конце: изометрический `get_viewport_screenshot` — визуальная проверка
   (две кровати, выдвинутые ящики, оба типа направляющих различимы).

Критерий готовности: все группы на месте с bbox по таблицам; скриншот
соответствует описанной сцене.

## 9. Материалы на одну кровать (закупка / раскрой)

| Материал | Кровать A | Кровать B |
|---|---|---|
| Щит сосна 18 | задняя 2056×420; торцевые 2× 920×420; царга 2020×130; фасады 2× 1022×270; изголовье 2× 478×900 | то же без изголовья; фасады 3× 680×270 |
| Фанера 18 | перегородка 920×360 — 1 шт | 2 шт |
| Фанера 12 | боковины 4× 750×240; перед/зад 2× 952×240 + 2× 967×240 | боковины 6× 750×240; перед/зад 6× 612×240 |
| Фанера 6 | дно 962×736 + 977×736 | дно 3× 622×736 |
| Доска 90×20 | ламели 13× 910 (~12 м пог.) | то же |
| Брусок 40×40 | 2× 2020 (~4.1 м) | то же |
| Направляющие 750 мм | 1 компл. боковых + 1 компл. скрытых | 3 компл. боковых |

Для стенок нужны полосы щита шириной 420 мм — кроить из щита 500/600 или
клеёный вариант. Карта раскроя листов — отдельная задача после выбора варианта.

## 10. Вне скоупа модели

Крепёж (конфирматы, уголки, шканты), ручки фасадов (выбор — открытый вопрос:
скобы или фрезерованный паз), подпятники, отделка (масло/лак), чертежи раскроя,
точная фрезеровка под скрытые направляющие.
