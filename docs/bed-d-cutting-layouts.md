## Раскрой щитов 2000×600

Схема щита: **основной ряд** — полоса высотой 420 по всей длине щита
(в неё встают стенки/задние); после отреза остаётся **полоса ~176**
(600 − 420 − пропил) — в неё встают царги (130) и планки (150), по три
штуки в длину (650+648+650 + 2 пропила = 1956 ≤ 2000). Фасады 680×270
ставятся **столбиком по два** (270+270+пропил = 544 ≤ 600) и могут делить
щит по длине максимум с одной 420-деталью (686+680 = 1370; вторая колонка
фасадов уже не помещается: +684 > 2000).

### 1 кровать — 6 щитов

| Щит | Основной ряд 420 | Полоса ~176 |
|---|---|---|
| 1 | 920 + 920 | царги 650 + 648 + 650 |
| 2 | 920 + 920 | планки 650 + 650 + 648 |
| 3 | 920 + 920 | планки 650 + 650 + 648 |
| 4 | 686 + 684 | запас |
| 5 | 686 + 2 фасада столбиком | — |
| 6 | 1 фасад | ~весь щит в остаток |

### 2 кровати — 11 щитов

| Щит | Основной ряд 420 | Полоса ~176 |
|---|---|---|
| 1–2 | 920 + 920 | царги 650 + 648 + 650 |
| 3–6 | 920 + 920 | планки 650 + 650 + 648 |
| 7 | 686 + 686 | запас |
| 8 | 684 + 684 | запас |
| 9–10 | 686 + 2 фасада столбиком | — |
| 11 | 2 фасада столбиком | ~2/3 щита в остаток |

### 3 кровати — 16 щитов

| Щит | Основной ряд 420 | Полоса ~176 |
|---|---|---|
| 1–3 | 920 + 920 | царги 650 + 648 + 650 |
| 4–9 | 920 + 920 | планки 650 + 650 + 648 |
| 10 | 686 + 686 | запас |
| 11 | 686 + 684 | запас |
| 12–15 | задняя (686/684) + 2 фасада столбиком | — |
| 16 | задняя + 1 фасад | остаток |

На щиты 12–16 идут оставшиеся задние: 686×3 + 684×2 (порядок неважен).

### Почему меньше не выходит

- В 420-ряд помещаются максимум **2 детали**: 920+920 = 1844 ✓, а три
  задние 686+684+686 + пропилы = 2064 > 2000.
- Фасад (ширина 270) рядом с 420-деталью по ширине щита не встаёт
  (420 + 270 = 690 > 600) — фасады делят щит только по длине, и тогда на
  щите максимум одна 420-деталь + 2 фасада; чисто фасадный щит вмещает 4.
- Перебор всех комбинаций этих паттернов даёт нижние границы 6 / 11 / 16 —
  карты выше их достигают.

## Раскрой фанеры 12 (лист 1525×1525)

Схема листа: **дно-ряд** — горизонтальная полоса высотой 750 (дна лежат
750 «в глубину листа»); **ряд 228** — полоса высотой 228, максимум 6 на
лист (6×228 + 5 пропилов = 1388 ≤ 1525), в ряд входят две детали
(750+750 = 1504 ✓; 601+599 и т.п.); **колонка** — вертикальная 228-полоса
справа от дон, в неё встают детали длиной до 750.

## Вариант Б — скрытые детали из фанеры 18

Идея: задние стенки, Γ-стенки стыков и планки пола снаружи не видны —
кроить их из фанеры 18 мм, оставив щиту только видимые детали (наружные
стенки, царги, фасады). Ящики (фанера 12) не меняются: те же 2 / 4 / 5
листов 1525×1525.

Видимость фанерных деталей: торец передней планки пола остаётся виден в
16-мм щели под фасадом (утоплен на 18 мм, в тени) — торец будет слоистым;
Γ-стенки и задние видны только изнутри камер при выдвинутых ящиках.
Пометка: у шлифованной фанеры допуск толщины (обычно −0,5…−1 мм от
номинала 18) — ширины камер чуть вырастут, зазор ящиков увеличится на доли
мм (для шариковых направляющих терпимо); фактическую толщину проверить при
покупке.

### Фанера 18 — листы 1525×1525

Плотный паттерн **П1** (~87 % листа): три детали 920×420 вертикальными
колонками (X 0–1268, Y 0–920) + планка вертикально (X 1272–1422, Y 0–650) +
две задние горизонтально в ряду Y 924–1344 (686+684 = 1374 ≤ 1525) + две
планки горизонтально в полосе Y 1348–1498 (650+650). Итого на листе:
3 детали 920×420 + 2 задние + 3 планки.

| Кроватей | Листы |
|---|---|
| 1 | Л1 = П1 (3Γ + 2 задние + 3 планки); Л2: 1Γ + 1 задняя + 3 планки |
| 2 | Л1–2 = П1; Л3: 2Γ колонками + 4 планки вертикально (X 848–1460, Y 0–650) + 2 задние (ряд Y 924–1344) + 2 планки (полоса Y 1348–1498) |
| 3 | Л1–4 = П1 (12Γ + 8 задних + 12 планок); Л5: 1 задняя + 6 планок |

На 3 кровати четырёх листов не хватает: потребовалось бы среднее заполнение
96,6 %, а лучший паттерн даёт 87 % — пятый лист неизбежен (уходит
полупустым, в задел).

### Фанера 18 — листы 2440×1220

Плотный паттерн **Л\*** (~78 %): два ряда по две детали 920×420
(Y 0–420 и 424–844, X 0–1844) + задняя вертикально (X 1848–2268, Y 0–686) +
до 6 планок горизонтально в полосе Y 848–1152 (два ряда по три:
650×3 + пропилы = 1958 ≤ 2440); при нужде ещё одна планка вертикально
X 2272–2422 (до кромки остаётся всего 18 мм).

| Кроватей | Листы |
|---|---|
| 1 | Л1 = Л* (4Γ + 1 задняя + 6 планок); Л2: 2 задние в ряд (686+684) |
| 2 | Л1–2 = Л*; Л3: 4 задние (ряд 686+684+686 = 2064 ≤ 2440 + ряд 684) |
| 3 | Л1–3 = Л* (12Γ + 3 задние + планки 6+6+3); Л4: 6 задних (два ряда по 686+684+686) + 3 планки (полоса Y 848–998) |

Нетто одной кровати (2,99 м²) больше площади одного листа (2,977 м²) на
0,017 м² — поэтому даже на одну кровать нужно 2 листа; 4 листа на 3 кровати
— оптимум по площади (75 %).

из площади (3,25 м² фанеры 12 на кровать) и достигается так:

| Кроватей | Лист | Состав |
|---|---|---|
| 1 | 1 | дно 625×750 ×2 + дно 623×750 + боковина 750×228 ×5 |
|  | 2 | боковина ×1 + все 6 деталей перед/зад |
| 2 | 1 | дно 625×750 ×3 + боковина ×5 |
|  | 2 | дно 625×750 + дно 623×750 ×2 + боковина ×5 |
|  | 3 | боковина ×2 + все 12 деталей перед/зад |
| 3 | 1–2 | дно 625×750 ×3 + боковина ×6 (каждый лист) |
|  | 3 | дно 623×750 ×3 + боковина ×6 |
|  | 4 | все 18 деталей перед/зад (601×228 ×12 + 599×228 ×6) |

На листах с тремя доньями они идут рядом вдоль стороны 2440:
3×750 + 2×4 = 2258 мм. Два ряда по три боковины дают общую занятую
высоту 625 + 4 + 228 + 4 + 228 = 1089 мм (для доньев 623 — 1087 мм).

Самые тесные — фанерные пары 750+750: при пропиле толще ~10 мм или листе
меньше номинала пересчитать.

## Остатки (задел)

- Щиты «920+920»: торцы ~156×420.
- Щиты с двумя задними: торцы ~620×420 + вся полоса ~176.
- Последний щит партии: от ~2/3 щита (2 кровати) до остатка при
  «задняя + 1 фасад» (3 кровати).
- Фанера: последний лист заполнен на 20–65 % в зависимости от партии.
